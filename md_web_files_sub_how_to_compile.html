<strong>准备</strong>

  • Python3: 在官网:https://www.python.org/downloads/ 下载安装。在window下默认安装会增加环境变量。
    在cmd窗口输入
    python -m pip install pip --upgrade
    python -m pip install clang-format # 可选安装，不影响编译
    python -m pip install clang-format # 可选安装，不影响编译
    下载失败原因：默认pip使用的是国外的镜像，下载的时候速度非常慢，网络也不稳定。因此，有很大的可能会出现失败的情况。
    pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
    pip config set install.trusted-host mirrors.aliyun.com
    # 使用以下命令查看配置
    pip config list

<strong>可选工具</strong>

pip install markdown
pip install python-markdown-math #公式

阿里云ECS Linux服务器图形化界面安装方法 Ubuntu

一、远程连接 二、登录后进入终端界面 三、更新服务器，安装图形化界面

sudo apt-get update sudo apt-get upgrade -y sudo apt-get dist-upgrade -y

执行apt-get upgrade会提示选择，输入y确认就好，确认后开始更新 四、开始安装图形化界面

sudo apt-get install ubuntu-desktop

五、更新完成，回到实例详情页重启 六、设置图形化界面，选择VNC方式远程登录

搭建git代码托管服务

sudo apt install openjdk-8-jdk

在 Fedora 34 上安装 GNOME 图形化界面的步骤如下：

执行以下命令更新系统：

sudo dnf upgrade –refresh 安装 GNOME 图形化界面：

sudo dnf groupinstall gnome-desktop 设置系统默认启动为图形化界面：

systemctl set-default graphical.target 重新启动系统：

sudo reboot
