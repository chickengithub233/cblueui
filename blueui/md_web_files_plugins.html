自动化测试脚本插件

支持javascript、Lua、ChaiScript等多种脚本语言。
产品提供了脚本执行接口。测试人员编写符合js语法的脚本。既可以完成测试

使用ControlObj关联控件

var obj = ControlObj("macdeskui");

使用ShellAttribute执行动作

obj.ShellAttribute("cmd", "value");

源文件生成工具

下载脚本

使用方法

# 查看帮助命令
python generate_code_cpp.py -h
#单个文件生成
python generate_code_cpp.py --file "abc.xml"
#目录生成
python generate_code_cpp.py --dir "abc"

皮肤文件工具

使用方法

# 查看帮助命令
python generate_skin_xml.py -h
# --theme 后面可以跟多个参数，一次性输出
python generate_skin_xml.py --input "ui文件目录" --output "输出目录" --theme "自定义主题名称"

语言生成工具

使用方法

#查看帮助命令
python generate_lang_xml.py -h
# --input后面可以跟多个参数
python generate_lang_xml.py --input "目录1" "目录2" "目录3" --output "输出目录"
