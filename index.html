简介

  CBlueUI是一款高质量、跨平台、跨框架的轻量可视化UI框架。支持Lua、js等多种脚本。实现自动化测试和存档。将开发和测试流程标准化。实现更高效的迭代开发速度。标准化的测试数据存档实现更可靠的溯源复盘。提供开发设计器和代码生成工具。一键从UI文件生成代码。即使是无经验的开发者也可轻松上手。

  目标: 致力解决业务对平台和技术框架的过度依赖。提供应用软件国产化迁移解决方案。在合理的成本控制下，加速企业数字化转型。
  原理: 采用自绘控件和抽象平台事件的的方式。整合主流UI框架的优点。以简单易用为出发点，重新开发可视化建模方案。
  适用领域: 客户端开发、可视化平台、矢量图形、组态开发，复杂交互场景开发。
  产生背景：国产操作系统社区成立以来。基础应用软件正朝着国产化适配的战略方向发展。操作系统是应用程序的基石。除了适配主流操作系统外，还应做好备选应急方案。才能为用户提供稳定可靠的服务。在危机来临时，守护用户核心利益，保住市场份额。

§. 功能

  • multi-platform for window / macos / linux / kylinos / embedded
    适配多种主流PC平台，以及嵌入式平台

  • multi-framework for win32 / cocoa / qt / gtk / sdl2 / wxWidgets
    多种兼容第三方框架可选方案，实现快速移植

  • multi-language for java / c# / python / html5
    多语言绑定开发（计划中...暂未实现）

  • Self developed software for rendering 3dmatplot charts
    完全自研实现基于软件渲染(未使用OpenGL)的3D Matplot科学图表控件

  • Optional functions OpenGL / OpenCV / ScanBarCode
    3D图形和机器视觉接口,需用户自己开发基于OpenGL的控件（可选项）

  • Support SVG format vector graphics
    支持SVG格式矢量图形

  • Supports mainstream scripting languages and custom formatting scripts
    自动化脚本，支持主流脚本语言和自定义格式脚本( lua / js ...)

  • Support for embedding CEF3 browser
    支持嵌入cef3浏览器（可选项）

  • Multiple reusable controls，Chart Visualization
    图表可视化能力。多种可复用的控件集。基础控件，仪表盘，时钟，金融K线，热力图，飞控姿态仪表，视频监控，扫码控件，组态，节点编辑器等等。无需二次开发。上手即用

  • Own self-developed text editor and syntax highlighting
    拥有完全自研的文本编辑器和语法高亮功能。

  • Parsable custom syntax format, generating syntax tree
    可解析自定义语法格式，生成语法树。比如解析类似C语言语法，金融指标公式语法TQL

  • Supports dynamic switching of styles and languages
    支持动态切换样式和语言

  • Provide UI designers and code generators. Improve development efficiency
    提供独立UI设计器和代码生成器。提高开发效率
